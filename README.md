# DocParser-AI

An intelligent document parsing and entity extraction tool built with **MiMo V2.5**.

## Features

- Document text input area with OCR simulation
- Named Entity Recognition (NER) extraction panel
- Entity type classification (Person, Org, Location, Date, etc.)
- Document classification with confidence tags
- Entity highlighting in text
- Export extracted entities as JSON
- Support for text, invoice, and contract document types

## Tech Stack

- Vanilla HTML5, CSS3, JavaScript (ES6+)
- Regular expression-based NER simulation
- Zero dependencies

## Getting Started

1. Open `index.html` in a modern web browser
2. Paste or type document text into the input area
3. Click "Parse Document" to extract entities
4. View extracted entities categorized by type
5. Toggle entity highlighting in the source text

## Entity Types

- **Person**: Names of individuals
- **Organization**: Company and institution names
- **Location**: Cities, countries, addresses
- **Date**: Dates and time expressions
- **Money**: Monetary values
- **Email**: Email addresses
- **Phone**: Phone numbers

Built with MiMo V2.5 AI assistance.
